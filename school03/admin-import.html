Stundenplan-Import

Ganze Stundenpläne auf einmal einfügen, statt jeden Termin einzeln anzulegen.

Format pro Zeile, mit Semikolon getrennt:
Datum;Zeit;Thema;Dozent;Raum
Beispiel: 2025-11-08;08:00 - 16:30;Fertigungstechnik;Nannt;Raum 1
Raum ist optional und kann weggelassen werden.
